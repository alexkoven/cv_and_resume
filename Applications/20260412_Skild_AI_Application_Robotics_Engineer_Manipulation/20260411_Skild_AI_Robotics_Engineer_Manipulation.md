## Overview

Skild AI develops "the world's first general purpose robotic intelligence that is robust and adapts to unseen scenarios without failing." The organization values demonstrated abilities and attitude over experience alone.

---

## Core Responsibilities

- Algorithm and technique design for robotic manipulation
- Hardware and software system architecture
- Controller and perception stack implementation for real-world deployment
- Production-level code maintenance in C++ and Python
- Collaboration with ML engineers to deploy learning-based models
- Software optimization for performance and scalability

---

## Required Qualifications

- Experience in designing robotic manipulation systems and deploying software on real robots
- Expertise in robotics fundamentals (kinematics, dynamics, control, motion planning, SLAM)
- Python proficiency plus deep learning libraries (PyTorch, TensorFlow, JAX)
- Systems understanding of ROS/ROS2 and robotics simulators
- BS/MS degree or equivalent practical experience

## Preferred Qualifications

- Conference publications in ML, robotics, or computer vision
- Custom robot hardware design experience
- Robot learning techniques (reinforcement learning, imitation learning)
- Computer vision and sensor expertise (LiDAR, 3D point clouds, object detection)
