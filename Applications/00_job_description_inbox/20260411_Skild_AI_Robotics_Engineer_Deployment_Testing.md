## Overview

Skild AI focuses on developing general-purpose robotic intelligence that adapts to unfamiliar scenarios. This position centers on advancing Skild AI's software through systematic testing protocols.

---

## Key Responsibilities

- Establish and maintain regression testing frameworks for company software
- Develop prioritized test plans considering impact, risk, and feasibility
- Investigate and resolve failures in complex testing and deployment scenarios
- Ensure robots meet deployment standards
- Convert production issues into reproducible test cases
- Partner with software and ML engineers to identify improvement opportunities

---

## Required Qualifications

- Degree in Computer Science, Robotics, or related fields (or equivalent experience)
- Proficiency with C, C++, Rust, Go, and Python
- Real-time embedded systems experience
- ROS/ROS2 platform experience
- Test management software experience
- Experience as a test engineer, robot deployment, software engineering fundamentals
- Machine learning integration in robotic systems
