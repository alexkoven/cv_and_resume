# Robotics Software Engineer — Skild AI

**Company:** Skild AI  
**Location:** Pittsburgh, San Francisco, Bengaluru  
**Salary:** $100,000 – $300,000 USD  
**Source:** https://job-boards.greenhouse.io/skildai-careers/jobs/4136373008

---

## Overview

Skild AI seeks engineers to develop software for robotic systems. The company focuses on "general purpose robotic intelligence that is robust and adapts to unseen scenarios" through machine learning at scale.

---

## Key Responsibilities

- Design and test software for navigation, SLAM, perception, manipulation, and control systems
- Write production-level C++ and Python code
- Integrate machine learning models with robotics engineers and researchers
- Deploy solutions at various sites with testing and deployment teams
- Optimize software for performance and reliability

---

## Required Qualifications

- Bachelor's degree or higher in Computer Science, Robotics, Engineering, or equivalent experience
- Proficiency in C++ or Python
- Prior experience deploying software on actual robots
- Strong understanding of robotic software modules and interfaces
- Technical expertise in one area: navigation, motion planning, SLAM, computer vision, or manipulation
- Knowledge of ROS/ROS2 or similar robotics middleware
- Software engineering fundamentals (algorithms, data structures, system design)
- Experience integrating machine learning into robotic systems

The company values demonstrated abilities and problem-solving attitude over industry experience alone.
