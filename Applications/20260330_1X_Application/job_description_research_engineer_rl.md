## About 1X

1X builds humanoid robots designed to work with people, addressing labor shortages and creating economic abundance.

## About the Role

This position focuses on teaching NEO humanoid robots new capabilities using reinforcement learning. The engineer operates across simulation and real-world settings to develop behaviors and deploy them into homes, enhancing robot safety, capability, and versatility.

## Responsibilities

- Manage complete engineering workflows from data preparation through deployment
- Train NEO on manipulation and locomotion tasks using RL
- Work with hardware teams to minimize sim-to-real performance gaps
- Coordinate with controls, QA, and data collection groups to release policies
- Implement trained skills in actual home environments

## Requirements

- Strong Python and/or C++ programming with build tools like Bazel
- PyTorch expertise
- Practical experience with Isaac Sim or MuJoCo
- Track record training RL policies for manipulation or locomotion
- Cross-functional collaboration capabilities
- Demonstrated sim-to-real gap solutions
