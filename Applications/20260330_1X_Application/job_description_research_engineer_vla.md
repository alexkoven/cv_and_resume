## About 1X

1X develops humanoid robots designed to work with people, addressing labor shortages and creating economic opportunities.

## About the Role

As a Research Engineer on the VLA team, you manage the complete development cycle of autonomous capabilities — from data evaluation and model architecture through system deployment and robot fleet performance monitoring. The focus is enabling robots to perform manipulation and navigation in new, unstructured settings.

## Responsibilities

- Own autonomous capability development end-to-end, spanning data analysis through deployment
- Train NEO robots for manipulation and navigation in novel environments
- Build evaluation metrics supporting scalable model development
- Apply advanced vision-language and generative models for robot action prediction
- Partner with control systems, QA, and data teams to operationalize reinforcement learning policies

## Requirements

- Strong Python programming; experience with build tools like Bazel
- Deep learning framework expertise, particularly PyTorch
- Working knowledge of simulation platforms (Isaac Sim or MuJoCo)
- Solid grasp of generalization principles in autonomous systems
- Demonstrated ability designing and validating performance metrics
- Cross-functional collaboration skills with engineering teams
